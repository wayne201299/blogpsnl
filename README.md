使用Hexo Next主題縮搭建的一個部落格平台，部署到GitHub Pages 做host
## 環境
* Node-16.13.0
* hexo: 6.1.0
* hexo-cli: 4.3.0

複製程式碼
```
git clone 
```
安裝第三方套件

hexo-cli（操作）
```
npm install -g hexo-cli
```
hexo-deployer-git（部署用）
```
npm install hexo-deployer-git --save
```

## 開發
啟動服務
```
hexo s
```

暫存清除
```
hexo clean
```

部署
```
hexo deploy
```
